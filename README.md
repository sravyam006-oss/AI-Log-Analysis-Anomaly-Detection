# AI Log Analysis and Anomaly Detection
## Overview
AI-powered Log Analysis and Anomaly Detection Web Application built using Python, Streamlit, Pandas, and Scikit-learn.
The system allows users to upload log files, automatically analyzes log entries, identifies suspicious patterns such as errors and failed activities, and uses the Isolation Forest machine learning algorithm to detect anomalies.
The application provides an interactive dashboard with metrics, visualizations, and anomaly reports, making log analysis easier and more accessible.
## Features
* Upload log files (.txt / .log)
* Automated log analysis
* Error and failed activity detection
* AI-based anomaly detection using Isolation Forest
* Interactive dashboard
* Metrics and visualizations
* Anomaly reporting
## Technologies Used
* Python
* Streamlit
* Pandas
* Scikit-learn
* Machine Learning
