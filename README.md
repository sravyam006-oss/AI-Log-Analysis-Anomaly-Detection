# AI-Log-Analysis-Anomaly-Detection
AI-powered Log Analysis and Anomaly Detection Web Application built using Python, Streamlit, Pandas, and Scikit-learn. The system allows users to upload log files, automatically analyzes log entries, identifies suspicious patterns such as errors and failed activities, and uses the Isolation Forest machine learning algorithm to detect anomalies. The application provides an interactive dashboard with metrics, visualizations, and anomaly reports, making log analysis easier and more accessible. Designed as a beginner-friendly project that combines machine learning, data analysis, and web application development.
